Publications & research

Peer-reviewed work demonstrating structured security engineering and analytical rigor.

Secure Blockchain Implementation Using RSA and SHA on FPGA

ECS Transactions (IOP Publishing)

Abstract

Blockchain technology is essential to secure storage, authenticate data, and protect information from misuse. Traditional cryptographic hashing methods (e.g., SHA-0, SHA-1) have limitations including computational overhead, collision risks, scalability constraints, and vulnerability to backtracking. This work proposes an encrypted blockchain block implemented on FPGA using a combined RSA and SHA approach. The design was synthesized and evaluated across Virtex-4, Virtex-5, and Spartan-6 platforms; Virtex-5 was selected based on resource efficiency (slice registers, LUT-FF pairs, memory). Security is strengthened through irreversible hashing and the inability to backtrack data, addressing prior limitations related to security, backtracking, memory usage, and collision risk.
